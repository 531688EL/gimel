### Commmands

print - `תצעק` <br>
exit - `יציאה` <br>
variable - `מכריז בזאת` <br>
function - `בראשית` <br>
for - `סעמק` <br>
while - `כוסעמק` <br> 
if - `וואלה-מה-אם` <br>
else - `וואלה-לא` <br>
